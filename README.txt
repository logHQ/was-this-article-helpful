== Description ==
Add "Was this article helpful?" at the end or start or both of article with thumbs up and thumbs down . Thumbs up would make to share and thumbs down would make to provide feedback to author via email

Features

1. Options to add "Was this article helpful?" at the End of the content.
2. Options to add "Was this article helpful?" at the Start of the content.
3. Options to add "Was this article helpful?" at the Start and End of the content.
4. Email Address for the Feedback from users can be changed.
5. Option to change Font Size and color for Thumbs Up and Thumbs Down Icon.
6. Text "Was this article helpful?" can be changed according to users choice.
7. Language Compatability.


__Shortcode__

Don't want to worry about HTML tags?  You can use a shortcode in your posts, pages to dispaly Article Feedback.

The shortcode to use is `[feedback_prompt]`.

**Examples**

`[feedback_prompt]`


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Use one of the two usage methods (HTML, Shortcode) within the content of your posts or pages.   

4. Use either the HTML or Shortcode methods inside your text widgets.
